# Todo App

## Live Demo
Check out the live app here: [Todo App Live](https://bhartidhotre.github.io/TodoApp/)

## Technologies
- JavaScript (DOM Manipulation)
- HTML
- CSS

## Description
A simple todo app allowing users to add, delete, and mark tasks as completed.

## Features
- Add, delete, and toggle tasks  
- Interactive DOM elements  

## Installation
1. Clone the repository:
   git clone https://github.com/bhartidhotre/TodoApp.git
2. Open `index.html` in your browser to view the project.
